# Odysseus kick-off exercise #

Exercise for working with the [Odysseus Framework](http://www.uni-oldenburg.de/informatik/is/forschung/projekte/odysseus/) to solve selected tasks of the [ACM DEBS 2013](http://www.orgs.ttu.edu/debs2013/index.php?goto=cfchallengedetails).
The documentation can be found [here](http://odysseus.informatik.uni-oldenburg.de:8090/display/ODYSSEUS/Odysseus+Home)

Was developed in summer semester 2014 at University of Bamberg.
